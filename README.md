# 🎴 神奇物种概率配置自动化系统

基于蒙特卡洛模拟的发卡概率测算工具，用于评估和优化卡牌收集类活动的概率配置。

## 功能概览

### 页面一：卡面选择页
- **卡池选择器**：展示10张卡（1张神奇卡 + 4张稀有卡 + 5张普通卡）
- **6个卡槽配置**：Week1（A×2 + B×1）和 Week2（C×2 + D×1）
- **蒙特卡洛模拟**：10万次迭代测算集齐率
- **推荐建议**：根据集齐率给出推荐等级（推荐/可接受/不推荐）
- **可视化图表**：集齐率对比柱状图

### 页面二：概率配置页
- **基础概率表配置**：调整10张卡的基础掉落概率
- **降权系数表配置**：设置不同持有数量下的概率系数
- **案例生成器**：模拟不同背包状态下的概率计算过程
- **概率分布展示**：降权前后的概率变化对比

## 技术栈

- **前端框架**：React 18 + TypeScript
- **UI组件库**：Ant Design 5.x
- **图表库**：Recharts
- **状态管理**：Zustand
- **构建工具**：Vite 5.x

## 核心算法

### 蒙特卡洛模拟流程

1. **生成14天日程**：每周随机打乱（5普通日 + 1稀有日 + 1神奇日）
2. **每日抽卡**：每天4次抽卡，根据当前背包状态计算概率
3. **应用降权系数**：A/B/C/D 卡根据持有数量应用不同系数
4. **归一化处理**：确保10张卡概率总和为100%
5. **统计结果**：记录 Week1 和 Week2 的集齐率

### 概率计算公式

```
Step 1: 获取基础概率
Step 2: 设置幸运卡概率 = 1.2%
Step 3: A/B/C/D 应用降权系数：P' = P × coefficient
Step 4: 填充卡第2张概率设为0
Step 5: 归一化：其他卡按比例分配剩余概率
Step 6: 验证总和 = 100%
```

### 降权系数设计原理

| 卡片类型 | 持有0张 | 持有1张 | 持有≥2张 |
|---------|--------|--------|---------|
| A/a (Week1) | 1.0 | 0.02 | 0 |
| B/b (Week1) | 1.0 | 0.02 | 0 |
| C/c (Week2) | 1.0 | 0.008 | 0 |
| D/d (Week2) | 1.0 | 0.008 | 0 |
| 填充卡 E-J | 1.0 | 0 | 0 |

**设计原理**：
- Week1 组合系数更高（2%），因为窗口期只有7天
- Week2 组合系数更低（0.8%），因为14天累积效应更强
- 填充卡确保只掉落1张，避免干扰组合收集

## 推荐阈值

| 最低集齐率 | 推荐等级 | 说明 |
|-----------|---------|------|
| ≥ 95% | 🟢 推荐 | 集齐率理想 |
| 80% - 95% | 🟡 可接受 | 有优化空间 |
| < 80% | 🔴 不推荐 | 用户难以完成 |
